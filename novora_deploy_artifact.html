Deploy it — $0, no API key

Live in three steps

The core math runs on-device — no paid model needed

The fast, protective checks are pure in-process math (Bayesian priors, a corroboration gate). ~5 ms, exactly $0.00 per call, zero network. That's why there's no API bill: the engine that ships is keyless.

Push to Vercel — it auto-deploys

The repo is git-connected. Merging to main builds and deploys the whole suite as serverless functions. No servers to run, no keys to configure for the free tier.

# from the repo root
git push        # merge to main → Vercel builds & deploys
# or a manual production deploy:
npm i -g vercel && vercel --prod
