Deploy Novora in 30 Minutes
Three steps from API key to live product. Vercel is free to start and scales to millions. Add your Anthropic API key and this file is a production-ready application.
Step 1 · 5 min
Get your Anthropic API key
Go to console.anthropic.com → Create account → API Keys → Create new key. Copy the key starting with sk-ant-api03-
→ console.anthropic.com
# Your key looks like:
sk-ant-api03-xxxx...
Step 2 · 5 min
Deploy to Vercel
Go to vercel.com → New Project → upload this HTML file. Or use the CLI to deploy from your computer in one command.
→ vercel.com — free tier
# CLI option:
npm i -g vercel
vercel --prod
Step 3 · 2 min
Add your API key
Vercel dashboard → Settings → Environment Variables → add ANTHROPIC_API_KEY with your key value. Redeploy. AI engine is live.
→ 30 minutes total to live
# Env variable name:
ANTHROPIC_API_KEY
Custom domain (5 extra minutes): Register novora.ai or novora.com at Namecheap (~$12/year) → Add to Vercel → Automatic SSL. Users see your domain, not vercel.app. Professional from Day 1.

API cost at scale: Each analysis costs ~$0.002–0.005 (Claude Sonnet). 1,000 free analyses/day = ~$3–5/day. Covered immediately by first 1–2 Pro subscribers. Every paying user at $12/month covers 2,400–6,000 free analyses.
For production at scale: Add a Node.js/Python backend (Vercel serverless functions) to keep the API key server-side, add rate limiting per user, connect Stripe for payments, and add a database (Supabase is free to start) for certificate storage. This file is the complete MVP — all nine products, AI-powered, deployable today.
